数据持久化-PlayerPrefs

PlayerPrefs

PlayerPrefs 的数据类型 类似于键值对存储 一个键对应一个值 (同一键名 不同类型就会覆盖数据)

提供了存储3种数据类型的方法 int float string

Set方法 直接调用只会把数据存到内存里 当游戏结束时 Unity会自动存储到硬盘中 但注意是要正常的游戏结束

Save调用该方法 就会立刻存到硬盘中

对数据的加密就是在原数据的基础上加上别的数据 在读取的时候减去这个数据 让别人不知道你的数据规则 他就无法在基础上得到你正确的数据
